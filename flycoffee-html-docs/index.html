Template Installation

1. Unzip the folder that you receive

2. Upload flycoffee-html contents in your public_html folder on your server

After uploading your site is finished and work. You can change text, images, video and other elements, just open with a text redactor the next files:
index.html - for homepage
contact.html - for contact page
and many more ...

How to change a menu

Open index.html or other file that you need, find the block "fly-site-navigation", and change or add an item menu

Note: If you need that this item menu appear on all pages, you need to add this new item menu in all your pages, contact.html, reservation.html, and others

How to add a new page for example photo gallery

Make a copy to photo.html and rename the file in gallery.html

Then open gallery.html and edit what you need, menus, images link, text

You can add a new item menu like in previous example

Note: This is an example, you can make what changes you need
